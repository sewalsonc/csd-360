Looping and Applying Math to Numbers 4 - 14

The instructions for this assignment are to create a webpage that will display a table that contains the base values of 4 through 14 and the results of math performed on each base number. The details of the table are listed below.

  1. The base values will be shown in the first column of the table.
  2. The square root of each base number will be displayed in the second column of the table.
  3. The base value to the power of 2 will be displayed in the third column of the table.
  4. The base value to the power of 3 will be displayed in the third column of the table.
