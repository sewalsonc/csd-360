JavaScript Fibonacci Numbers

by Chris Sewalson

Looping with JavaScript

This page displays the first 30 Fibonacci numbers. The Fibonacci sequence is the consecutive sum of the preceding 2 numbers. Such as 0 + 1 is 1, then 1 + 2 is 3, then 3 + 2 is 5 and so on and so forth.

We are to do the following:
  1. Declare the variables
  2. Display the page title
  3. Display the first and second values of 1
  4. Start a for loop and a counter with a value of 3
  5. Use less than or equal to 30 for the loop stop condition
  6. Assign a totalizer variable with sum of the preceding 2 values
  7. Output the values of the counter and totalizer
  8. Assign the first variable the value of the first number
  9. Assign the second variable the value of the totalizer
  10. Increment the counter
  11. Test the loop again
  12. Use the document.write to display the output
